🛒 Small E-Commerce Project
This project is a functional E-commerce application designed with a focus on clean code architecture and modern JavaScript patterns.

The goal of this app isn't just the final UI—it’s about the technical journey and mastering the core concepts required for scalable web development. 🚀

--------------------------------------------------------------------------------
--------------------------------------------------------------------------------

🧠 Technical Focus
During the development of this application, the primary focus is on implementing:

🔄 Promises & Asynchrony Mastering asynchronous data handling to ensure a smooth, non-blocking user experience when fetching products or processing orders.

🏗️ Object-Oriented Programming (OOP) Organizing the codebase using Classes and Objects. This ensures that the cart, product list, and user interactions are modular, reusable, and easy to maintain.

--------------------------------------------------------------------------------
--------------------------------------------------------------------------------

✨ Key Features
Dynamic Product Listing: Fetched asynchronously from the mock server.

Structured Logic: Every component is built using OOP principles.

Mock Backend: A fully functional data environment using db.json.

--------------------------------------------------------------------------------
--------------------------------------------------------------------------------

🛠️ Tech Stack
Logic: JavaScript (ES6+)

Data: JSON Server

Patterns: OOP & Promises
